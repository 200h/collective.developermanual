==========================
Internationalization
==========================

Plone has three different internationalization subsystems

* Translating user interface layer by using gettext - zope.i18n, zope.i18nmessageid packages

* Adapting locale specific settings for the site - plone.i18n package

* Managing multilingual content - Products.Linguaplone add-on product
